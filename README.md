This project integrates an Aadhaar QR Code Scanner with a Blockchain-based Voting System. Users can scan their Aadhaar QR code to authenticate and securely cast votes on the Ethereum blockchain. The QR data is validated and stored in MongoDB, ensuring a tamper-proof, decentralized voting process.

Features:
Aadhaar QR Code Scanning: Real-time QR scanning using React and react-qr-reader. //cancelled due to aadhar encryption
Blockchain Integration: Votes are recorded immutably on the Ethereum blockchain using Solidity smart contracts.
Secure Database: Aadhaar data is securely stored in MongoDB for authentication.
Full-Stack Implementation: Built using React.js (frontend), Node.js (backend), and Hardhat/Ganache (blockchain testing).
